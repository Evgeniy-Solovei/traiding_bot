## 🚀 Как использовать

### Быстрый старт (5 минут)

```bash
# 1. Установка
cd /home/user/webapp
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. Настройка БД
sudo -u postgres psql -c "CREATE DATABASE trading_bot;"
sudo -u postgres psql -c "CREATE USER user WITH PASSWORD 'password';"
sudo -u postgres psql -c "GRANT ALL PRIVILEGES ON DATABASE trading_bot TO user;"

# 3. Генерация ключа шифрования
./generate_key.py

# 4. Настройка .env
cp .env.example .env
nano .env  # Добавьте TELEGRAM_BOT_TOKEN и ENCRYPTION_KEY

# 5. Миграции
python manage.py migrate

# 6. Запуск
./start.sh

# 7. Используйте бота в Telegram!
```

## 📊 Что делает бот

1. **Каждую минуту**:
   - Получает свечи с Bybit для каждой активной торговой пары
   - Рассчитывает индикаторы (EMA, RSI, Williams %R, ATR)
   - Определяет тренд
   - Строит ценовой канал
   - Ищет сигналы на вход

2. **При обнаружении сигнала**:
   - Проверяет, нет ли уже открытой позиции
   - Рассчитывает размер позиции на основе риска
   - Валидирует параметры
   - Устанавливает плечо
   - Открывает рыночный ордер
   - Сохраняет сделку в БД
   - Отправляет уведомление пользователю

3. **Каждые 30 секунд**:
   - Проверяет все открытые позиции
   - Сравнивает текущую цену со стоп-лоссом и тейк-профитом
   - При достижении - закрывает позицию
   - Рассчитывает PnL
   - Обновляет статистику
   - Отправляет уведомление

4. **Каждый час**:
   - Обновляет статистику торговли для всех пользователей

## ⚠️ Важные замечания

### Безопасность
- ✅ API ключи шифруются перед сохранением
- ✅ ENCRYPTION_KEY хранится в .env (не в коде!)
- ✅ .env добавлен в .gitignore
- ⚠️ Используйте IP whitelist на Bybit для mainnet

### Риски
- ⚠️ Торговля фьючерсами = высокий риск
- 💡 Начните с testnet для тестирования
- 💡 Используйте риск 1-2% на сделку
- 💡 Бот НЕ гарантирует прибыль
- 💡 Следите за балансом и статистикой

### Ограничения
- Только Bybit
- Только фьючерсы USDT
- Одна стратегия (параметры можно менять в админке)
- Таймфрейм по умолчанию: 5m
